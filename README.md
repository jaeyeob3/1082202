# 한국주식 펀더멘탈 투자등급 사이트

회사명(예: `삼성전자`) 또는 종목코드(예: `005930`)를 입력하면, 밸류에이션과 퀄리티를
**시장 전체 분포 대비 백분위**로 평가해 **적극매수 / 매수 / 중립 / 매도 / 적극매도** 5단계
등급을 산출하는 로컬 웹앱입니다.

## 실행 방법

데이터 수집을 위해 **인터넷이 연결된 본인 PC**에서 실행해야 합니다(KRX·네이버 금융 접속).

### Windows (가장 간단)
`run_windows.bat` 더블클릭 → 패키지 자동 설치 후 브라우저가 자동으로 열립니다.

### Mac / Linux
터미널에서 한 번만 실행 권한을 준 뒤 더블클릭 또는 실행:
```bash
chmod +x run_mac_linux.sh
./run_mac_linux.sh
```

### 직접 실행 (공통)
```bash
pip install -r requirements.txt
streamlit run app.py
```
실행 후 브라우저에서 http://localhost:8501 이 자동으로 열립니다.
첫 화면의 **인기 종목 버튼**(삼성전자·SK하이닉스·현대차·NAVER·카카오·KB금융)을
누르면 바로 분석되고, 직접 회사명/코드를 입력할 수도 있습니다.

## 평가 방식

| 분류 | 팩터 | 방향 | 기본 가중치 |
|------|------|------|------------|
| 밸류 | PER | 낮을수록 ↑ | 18% |
| 밸류 | PBR | 낮을수록 ↑ | 17% |
| 밸류 | 배당수익률 | 높을수록 ↑ | 10% |
| 퀄리티 | ROE | 높을수록 ↑ | 16% |
| 퀄리티 | EPS 성장률(YoY) | 높을수록 ↑ | 9% |
| 퀄리티 | 이익의 질(흑자) | 흑자 ↑ | 6% |
| 퀄리티(심화·DART) | 부채비율 | 낮을수록 ↑ | 10% |
| 퀄리티(심화·DART) | 영업이익률 | 높을수록 ↑ | 10% |
| 퀄리티(심화·DART) | 매출성장률(YoY) | 높을수록 ↑ | 5% |
| 모멘텀 | 6M 수익률·52주 위치 | 강세 ↑ | 7% |

- 각 팩터는 **시장 전체 종목 분포 대비 백분위**로 0~100점 환산 → "전문 리서치식 상대평가".
- 종합점수: **75↑ 적극매수 · 60↑ 매수 · 40↑ 중립 · 25↑ 매도 · 그 미만 적극매도**.
- 가중치는 왼쪽 사이드바 슬라이더로 실시간 조정 가능(가치투자/성장투자 성향 반영).
- **ROE는 `PBR/PER` 항등식으로 도출한 근사치**입니다(별도 키 불필요).
- **DART 인증키를 입력하면** 부채비율·영업이익률·매출성장률 3개 심화 퀄리티 팩터가 더해져
  7개 → **10개 팩터**로 평가합니다. 키가 없으면 자동으로 7개 팩터만 사용(가중치 자동 재정규화).
  (심화 3개는 절대 기준 점수, 나머지는 시장 백분위 점수)

## DART 연동 (선택, 권장)
1. https://opendart.fss.or.kr 에서 **무료 인증키** 신청(이메일 인증, 즉시 발급).
2. 앱 왼쪽 사이드바 **"DART 인증키"** 칸에 40자리 키를 붙여넣고 분석.
3. 결과의 핵심지표에 부채비율/영업이익률/매출성장률/순이익률이 추가로 표시됩니다.
   - 연간 사업보고서(연결재무제표 우선) 기준이며, 최근 공시 연도부터 자동 탐색합니다.

## 파일 구성
- `app.py` — Streamlit 웹 UI(가중치 슬라이더 + DART 키 입력)
- `scoring.py` — 점수화·등급 엔진(순수 함수, 네트워크 불필요)
- `data_source.py` — KRX/네이버 데이터 수집
- `dart.py` — DART 오픈API 재무제표 수집·비율 산출
- `test_scoring.py` — 엔진 검증 테스트 (`python test_scoring.py`)
- `requirements.txt`
- `run_windows.bat` — 윈도우 원클릭 실행
- `run_mac_linux.sh` — 맥/리눅스 원클릭 실행

## 한계
